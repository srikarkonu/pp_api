Welcome to Real Estate Properties API Home Page

Use the following endpoints in the URL to obtain desired results:

  • /filter_properties/?city=Hyderabad - Filter properties by city
  • /filter_properties/?locality=Gachibowli - Filter properties by locality
  • /filter_properties/?pincode=500032 - Filter properties by pincode
  • /filter_properties/?bhk=2 - Filter properties by number of bedrooms
  • /filter_properties/?area=1000 - Filter properties by area

Modify the parameters as needed to refine your search.
